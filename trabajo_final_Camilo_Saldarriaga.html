Introducción¶

El objetivo de este análisis es caracterizar las relaciones entre algunos indicadores macroeconómicos y los costos y gastos de ventas de las empresas colombianas vigiladas por la SuperSociedades.

Este analisis se hace con datos de periodisidad Anual, debido a que es la peridisidad que maneja la Superintendencia de Sociedades. Buscamos identificar un sector en particular, usando la categoria CIIU (Clasificación Industrial Internacional Uniforme), que tenga por lo menos 20 empresas, para hacer el analisis a nivel de sector ecónomico.

Los resultados de este análisis estan incluidos en un repositor de GitHub el cual esta públicamente accesible en el siguiente enlace:

https://github.com/juancsaldarriaga/Informe_sector_consultoria_MAE

Para la descarga de datos lo que hice fue que consulte diferentes en bases de datos oficiales como:

  • Datos abiertos del gobierno nacional (www.datos.gov.co)
  • DANE (https://www.dane.gov.co/index.php/estadisticas-por-tema)
  • Banco Mundial (https://datos.bancomundial.org/)
  • Cepal (https://estadisticas.cepal.org/cepalstat/WEB_CEPALSTAT/estadisticasIndicadores.asp?idioma=e),
  • Banco de la Republica (https://www.banrep.gov.co/es/catalogo-estadisticas-disponibles).

Fue dificil ubicar algunos de los archivos debido a que las fuentes oficiales puede tener enlaces caidos y rotos, por lo que me toco buscar una agencia más internacional que recopilara la información macroeconomica para varios paises, como la Cepal y el Banco Mundial.

Resultados¶

Se utiliza un modelo de regresión lineal para modelar la relación entre las variables PIB, Inflación, Desempleo, Tasa de Cambio-TRM y los costos y gastos de ventas.

Aqui se presenta el modelo lineal que mejores resultados obtuvo usando las variables macroeconomicas disponibles. El que mejor se comporto es el que utiliza como variable dependiente el logaritmo natural de los costos $log\_costos$, y los regresores: la tasa de desempleo $Desmpleo$ y la tasa de inflación $inflacion$. $𝑙𝑜𝑔\_𝑐𝑜𝑠𝑡𝑜𝑠 = 𝐷𝑒𝑠𝑚𝑝𝑙𝑒𝑜 + 𝑖𝑛𝑓𝑙𝑎𝑐𝑖𝑜𝑛 + E_i$
