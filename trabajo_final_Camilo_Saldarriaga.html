Introducción¶

El objetivo de este anlisis es caracterizar las relaciones entre algunos indicadores macroeconmicos y los costos y gastos de ventas de las empresas colombianas vigiladas por la SuperSociedades.

Este analisis se hace con datos de periodisidad Anual, debido a que es la peridisidad que maneja la Superintendencia de Sociedades. Buscamos identificar un sector en particular, usando la categoria CIIU (Clasificacin Industrial Internacional Uniforme), que tenga por lo menos 20 empresas, para hacer el analisis a nivel de sector ecnomico.

Los resultados de este anlisis estan incluidos en un repositor de GitHub el cual esta pblicamente accesible en el siguiente enlace:

https://github.com/juancsaldarriaga/Informe_sector_consultoria_MAE

Para la descarga de datos lo que hice fue que consulte diferentes en bases de datos oficiales como:

  • Datos abiertos del gobierno nacional (www.datos.gov.co)
  • DANE (https://www.dane.gov.co/index.php/estadisticas-por-tema)
  • Banco Mundial (https://datos.bancomundial.org/)
  • Cepal (https://estadisticas.cepal.org/cepalstat/WEB_CEPALSTAT/estadisticasIndicadores.asp?idioma=e),
  • Banco de la Republica (https://www.banrep.gov.co/es/catalogo-estadisticas-disponibles).

Fue dificil ubicar algunos de los archivos debido a que las fuentes oficiales puede tener enlaces caidos y rotos, por lo que me toco buscar una agencia ms internacional que recopilara la informacin macroeconomica para varios paises, como la Cepal y el Banco Mundial.
